# Challenge #5: Blog Post
Write a blog post about a topic you care about (short; ~500 words). Turn that blog post into a webpage using HTML and CSS. The webpage should include your post content, a title, and and image. Commit and publish your blog to Netlify and upload the public Netlify and GitHub URLs to the C5: Dropbox. You do not need to create a Figma design/PNG export for this challenge.
